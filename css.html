Basic CSS Project

A project to help practice common CSS properties and patterns.

Project Background

This purpose of this project was to help me demonstrate mastery of basic CSS tecnhiques by using CSS to to position and style the elements of an HTML page to create an elegant visual presentation of the poem "Kubla Khan" by Samuel Taylor Coleridge. I was encouraged not to edit the HTML file.

  • Basic Selectors: use basic CSS selectors for HTML elements, classes and IDs to control the appearance of the content
  • External Spreadsheets: create and/or utilize external spreadsheets and link them into an HTML document
View Live Site

Requirements

  • Fork the source repository into my personal GitHub account
  • Clone the source repository into my personal development enviornment for a starter "boilerplate"
  • Complete the build tasks as outlined in the README.md file in the repository
  • Demonstrate effective use of the following CSS properties
    • Background
    • Font-family
    • Color
    • Margins
    • Padding
    • Border

Actions & Outcomes

The portions of this assignment that were particularly difficult were understanding the how to format the “box model” layout and create CSS properties that changed the specific portions of the HTML file that I was looking to alter.

Conclusions

I enjoyed the assignment overall very much because it was a learning experience for me from start to end and at the end, I saw a completed webpage that I was proud of. I feel that my approach to the reading and instructional video best set me up for this assignment and wouldn’t approach the assignment any differently.
