Multi-View App

A project for practicing with user input and multiple routes using vue router-links.

Project Background

This project tasked me with creating a form to collect user inputs, form validation, and v-for loops and a $router.push statement to a secret page.

  • State Management: handle routing user requests within the app, including URL and State management and related systems as needed
  • Importing Dependencies: use multiple JavaScript modules together with proper import statements and well-considered file management
  • Event Handling: respond to events within the webapp
  • Handling Forms and Input: create HTML forms using the tools of a JavaScript framework to specify behaviors and coordinate data in the system so users can successfully input information

Actions & Outcomes

This project required me to create an application that used more than one view or "route". I had to break it down into the following milestones to help me get through it and ensure nothing was missed.

  • Create an HTML form
  • Create the form validation (Using v-show and a simple submit button)
  • Add the views to the routes definition on the index.js
  • Create both an additional Secret page and Survey page
